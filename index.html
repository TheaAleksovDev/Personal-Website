PORTFOLIO

Nodinity

Nodinity is a platform for developers to showcase their portfolios in a visually interactive way.

Users can create graphs with their username at the center, surrounded by nodes for categories like projects, education, and personal achievements. This project demonstrates my skills in Angular. It provides a creative and engaging way for developers to highlight their work and connect professionally. I built Nodinity using Angular for the frontend and Sigma.js for the graph visualization. Firebase was used for real-time data storage and user authentication, ensuring a smooth and secure experience.

Intellectum - mobile web game

Intellectum is a quiz game aimed at making learning enjoyable, where users can test their knowledge across diverse categories while earning rewards and unlocking themed characters inspired by real-life figures.

This project showcases my proficiency in frontend development, backend integration, and creative design, offering users an immersive and enjoyable gaming experience.

I spearheaded the development of Intellectum, a captivating quiz game optimized for mobile web platforms. Using React and TypeScript, I engineered the frontend interface for seamless user interaction. Backend integration was achieved through Firebase, ensuring real-time data synchronization and robust user authentication.

Additionally, I took charge of the creative direction, crafting all graphics and characters from scratch using Photoshop and Illustrator. By incorporating gamification elements such as rewards and themed character unlocks, I enhanced user engagement and retention.

SpotSquad - web app

The core functionality of SpotSquad revolves around facilitating seamless collaboration and scheduling for users scattered across different towns. By leveraging Firebase for backend support, users can create a shared server where they connect and synchronize their calendars .

SpotSquad exemplifies my expertise in building cross-platform solutions for modern needs. Developed using React Native, JavaScript, and Firebase, SpotSquad addresses the challenge of coordinating meetings among individuals constantly on the move.

The app's intuitive interface allows users to view each other's schedules and locations, making it effortless to coordinate appointments and live meetings. With customizable town and member preferences, users can filter and prioritize relevant information, ensuring efficient planning regardless of their geographical locations.
